RabaRush: Full-Stack Morabaraba Platform

An online, multi-user web application featuring a scalable Node.js and Express backend, relational MySQL database architecture, and real-time state management pipelines.

Implementation

This project focuses on the design, engineering, and deployment of the backend service layer and secure transaction pipelines for RabaRush, a synchronized online web platform for the traditional game Morabaraba. Built utilizing Node.js and Express.js, the backend acts as a highly modular centralized engine responsible for routing, handling concurrent client requests, and enforcing strict server-side validation boundaries. The system isolates business logic from the presentation layer to protect user interactions, prevent data tampering, and manage state transitions safely during multi-user gameplay sessions.

Data management and schema integrity are handled through a persistent relational MySQL database system, which securely tracks real-time data fields including customer registration profiles, comprehensive match histories, and player win-loss tracking. To govern account security and satisfy technical data safety requirements, a robust authentication pipeline was engineered. Passwords undergo computational cryptographic hashing via bcrypt before database insertion, and secure, stateless session tracking is established by provisioning JSON Web Tokens (JWT) to authorize subsequent actions across protected gaming APIs.

In addition to core account services, the backend safely facilitates the platform's multi-tenant features, managing user coin balances and processing transactional systems securely to prevent race conditions during real-time multiplayer engagements. To bridge the server architecture with the user-facing application, a clean frontend login component was developed in vanilla JavaScript to manage payload delivery, parse server verification tokens, and handle asynchronous user session lifecycles. Designed in alignment with structural principles outlined in the SWEBOK framework, the complete ecosystem underwent rigorous regression testing using Postman to validate api endpoint durability, verify edge cases, and ensure a highly secure, reliable MVP foundation.
